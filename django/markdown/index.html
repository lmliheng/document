如何获取网页文章Markdown形式

1.在CSDN博文页面点击右键，选择“检查”（Google浏览器为例）。

2.在查看器中搜索article_content，找到对应内容，点击…复制为outerHTML。

3.打开网址https://tool.lu/markdown/，点击 HTML2MD ，粘贴html代码，转换成Markdown。 参考文献
