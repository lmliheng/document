欢迎

如需完整文档，请访问mkdoc文档

命令

  • pip install mkdocs - pip安装mkdocs库
  • mkdocs new [dir-name] - 创建一个新项目。
  • mkdocs serve - 启动实时重新加载文档服务器。
  • mkdocs build - 构建文档站点。
  • mkdocs -h - 打印帮助信息并退出。

项目布局

mkdocs.yml    # 配置文件
docs/
    index.md  # 文档主页。
    ...       # 其他 Markdown 页面、图片和其他文件。

mkdocs.yml基本配置

 site_name: Django杂记
 site_url: https://example.com/
 nav:
     - 主页: index.md
     - mkdocs笔记: mkdocs.md
     - 副页: 
           - '分页1': 'one.md'
           - '分页2': 'two.md'

theme: readthedocs
