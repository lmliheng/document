### GFM
`GFM`（GitHub Flavored Markdown）是一种扩展了标准 Markdown 语法的格式支持一种特殊的任务列表语


1. 对于未完成任务，使用 - [ ]
2. 对于已完成任务，使用 - [x]
示例：
```bash
## My Task List

- [ ] Task 1 
- [x] Task 2 
- [ ] Task 3 
```
