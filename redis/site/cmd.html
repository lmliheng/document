  • 命令

检查 redis-cli 是否可用：

which redis-cli

打开Redis 客户端

redis-cli

输入密码提示:

NOAUTH Authentication required.

输入密码：AUTH 密码

检测 redis 服务是否启动

$ redis-cli
redis 127.0.0.1:6379>
redis 127.0.0.1:6379> PING

PONG

重启redis服务

sudo systemctl restart redis

info命令:

info

info命令用于获取Redis服务器信息。
