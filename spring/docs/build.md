## Maven使用
下载解压`apache-maven-3.9.6`
```
├─bin
├─boot
├─conf
│  └─logging
└─lib
    ├─ext
    │  ├─hazelcast
    │  └─redisson
    └─jansi-native
        └─Windows
            ├─x86
            └─x86_64
```
```bash
maven -V
```
```bash
maven clean install -U
```
